Steps to run the code:

Step 1: run team2box.py to get the embedding of teams
Step 2: run ExpertsEmbeding.py to get embedding of questions and experts
Step 3: download and run https://github.com/AdeDZY/K-NRM using the data in datasetname/knrmformat/ directory and save the model
Step 4: run runknrm.py to get top k similar quetions for test questions
Step 5: run team2box_teamformation.py to form the teams and get the results
